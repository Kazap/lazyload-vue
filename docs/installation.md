## YARN

```sh
yarn add lazyload-vue
```

## NPM

```sh
npm install lazyload-vue --save
```

When used with a module system, you must explicitly install LazyloadVue via Vue.use():

```js
import Vue from 'vue'
import LazyloadVue from 'lazyload-vue'

Vue.use(LazyloadVue)
```
